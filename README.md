# Building LLM

A project for building Large Language Models from scratch.

## Overview

This repository contains resources, code, and materials for understanding and constructing LLMs from the ground up.

## Project Structure

- `course_outputs/` - Course materials and outputs
- `skills/` - Skill-based learning resources
- `animation.py` - Animation utilities for visualizations
- `media/` - Generated media files

## Getting Started

Clone this repository:
```bash
git clone https://github.com/sanjabh11/LLM_from_scratch.git
```
